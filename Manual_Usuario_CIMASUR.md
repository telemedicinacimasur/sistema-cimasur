# Manual de Usuario: Sistema de Gestión CIMASUR

## 1. Introducción
Este manual describe el uso de los módulos del Sistema de Gestión CIMASUR, diseñado para la administración integral de telemedicina.

## 2. Acceso al Sistema
Para ingresar, acceda a la URL de la aplicación y proporcione sus credenciales (correo electrónico y contraseña).

## 3. Módulos del Sistema

### 3.1. Dashboard (Panel Principal)
Es la pantalla de inicio tras iniciar sesión. Muestra un resumen general de la actividad y acceso rápido a los módulos asignados según su perfil.

### 3.2. Módulo de Laboratorio
*   **Acceso:** Roles Admin y Lab.
*   Permite la gestión de pruebas, carga de resultados de laboratorio y seguimiento de muestras.

### 3.3. Módulo de CRM (Gestión de Clientes)
*   **Acceso:** Roles Admin y CRM.
*   Facilita el seguimiento de pacientes, historial de contactos y gestión de prospectos.

### 3.4. Módulo Escuela (Gestión Escolar)
*   **Acceso:** Roles Admin y School.
*   Control de información académica, reportes y seguimiento escolar vinculado a los programas de salud.

### 3.5. Módulo de Administración
*   **Acceso:** Solo Rol Admin.
*   Gestión completa de usuarios del sistema, creación de nuevas cuentas, edición de roles y configuraciones generales.

## 4. Finalización
Para cerrar la sesión, utilice el botón de "Cerrar sesión" ubicado comúnmente en la barra de navegación lateral o superior.
